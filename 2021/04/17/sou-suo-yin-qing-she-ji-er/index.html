基于Elasticsearch的电影搜索引擎设计(二)

在第一部分中，我们从IMDB网站上搜刮了数据，并将其传入到Elasticsearch中。

在本文中，我们将构建搜索引擎前端，以及两种不同的检索模型，用于搜索电影数据。

Tools 准备

我们的Python Flask应用程序将成为我们的Web框架，并且我们将使用jinja模板来呈现html文件。

在PyCharm中创建新的项目，选择Flask (我使用的是Python3.8，Jinja2)。

检索模型的构建

对于我们的app.py文件：

from elasticsearch import Elasticsearch, helpers
from flask import Flask, render_template, request
from SearchForm import SearchTitleForms

es = Elasticsearch('http://localhost:9200')
app = Flask(__name__)
app.secret_key = 'my secret key'


@app.route('/', methods=['GET'])
@app.route('/search', methods=['POST', 'GET'])
def search():

    # The function to capture the fields as a new index dataset
    def gen_bm25_Data(query_list):
        for movie in query_list:
            name = movie.get('movie_name', None)
            types = movie.get('movie_type', None)
            score = movie.get('movie_score', None)
            director = movie.get('director', None)
            stars = movie.get('stars', None)
            synopsis = movie.get('movie_synopsis', None)
            bm25f = movie.get('BM25F', None)

            yield &#123;
                "_index": "bm25_dataset",
                "_source": &#123;
                    "movie_name": name,
                    "movie_type": types,
                    "movie_score": score,
                    "director": director,
                    "stars": stars,
                    "movie_synopsis": synopsis,
                    "BM25F": bm25f,
                &#125;,
            &#125;

    form = SearchTitleForms()

    if request.method == 'POST':

        # Search in BM25 model
        if form.bm25_search.data:
            q = request.form['q']
            print("We are looking for : ", q, " In elastic search ")

            body = &#123;
                "size": 10000,
                "query": &#123;
                    "multi_match": &#123;
                        "query": q,
                        "fields": ["movie_name", "movie_type", "movie_score", "director", "stars", "movie_synopsis"]
                    &#125;
                &#125;
            &#125;

            query = es.search(index='imdb_dataset', body=body)
            results = query['hits']['hits']
            for movie in results:
                movie['_source']['query_score'] = movie['_score']

            return render_template('es_searchengine.html', form=form, response=results)

        # Create a new dataset with the determined weight object and weight value
        elif form.dataset.data:
            select = request.form["select"]
            weight = int(request.form["weight"])

            m_name_weight = False
            m_type_weight = False
            m_score_weight = False
            director_weight = False
            stars_weight = False
            m_synopsis_weight = False

            if select == 'Title':
                m_name_weight = True
            if select == 'Type':
                m_type_weight = True
            if select == 'Score':
                m_score_weight = True
            if select == 'Director':
                director_weight = True
            if select == 'Actors':
                stars_weight = True
            if select == 'Synopsis':
                m_synopsis_weight = True

            q = request.form['q']
            print("We are looking for : ", q, " In elastic search ")

            body = &#123;
                "size": 10000,
                "query": &#123;
                    "match_all": &#123;
                    &#125;
                &#125;
            &#125;
            query = es.search(index='imdb_dataset', body=body, sort="_score")
            results = query['hits']['hits']

            query_list = []
            for movie in results:
                query_list.append(movie['_source'])
            for movie_source in query_list:
                bm25f = ""

                if m_name_weight == True:
                    for i in range(weight):
                        bm25f = bm25f + str(
                            'None' if movie_source['movie_name'] is None else movie_source['movie_name']) + ' '
                else:
                    bm25f = bm25f + str(
                        'None' if movie_source['movie_name'] is None else movie_source['movie_name']) + ' '

                if m_type_weight == True:
                    for i in range(weight):
                        bm25f = bm25f + str(
                            'None' if movie_source['movie_type'] is None else movie_source['movie_type']) + ' '
                else:
                    bm25f = bm25f + str(
                        'None' if movie_source['movie_type'] is None else movie_source['movie_type']) + ' '

                if m_score_weight == True:
                    for i in range(weight):
                        bm25f = bm25f + str(
                            'None' if movie_source['movie_score'] is None else movie_source['movie_score']) + ' '
                else:
                    bm25f = bm25f + str(
                        'None' if movie_source['movie_score'] is None else movie_source['movie_score']) + ' '

                if director_weight == True:
                    for i in range(weight):
                        bm25f = bm25f + str(
                            'None' if movie_source['director'] is None else movie_source['director']) + ' '
                else:
                    bm25f = bm25f + str('None' if movie_source['director'] is None else movie_source['director']) + ' '

                if stars_weight == True:
                    for i in range(weight):
                        bm25f = bm25f + str('None' if movie_source['stars'] is None else movie_source['stars']) + ' '
                else:
                    bm25f = bm25f + str('None' if movie_source['stars'] is None else movie_source['stars']) + ' '

                if m_synopsis_weight == True:
                    for i in range(weight):
                        bm25f = bm25f + str(
                            'None' if movie_source['movie_synopsis'] is None else movie_source['movie_synopsis']) + ' '
                else:
                    bm25f = bm25f + str(
                        'None' if movie_source['movie_synopsis'] is None else movie_source['movie_synopsis']) + ' '

                length = len(bm25f.split())
                if length < 100:
                    for i in range(100-length):
                        bm25f = bm25f + ' None'
                movie_source['BM25F'] = bm25f

            helpers.bulk(es, gen_bm25_Data(query_list))

            return render_template('es_searchengine.html', form=form)

        # Search in BM25F model
        elif form.bm25f_search.data:

            q = request.form['q']

            body2 = &#123;
                "size": 10000,
                "query": &#123;
                    "match": &#123;
                        "BM25F": q
                    &#125;
                &#125;
            &#125;

            query = es.search(index="bm25_dataset", body=body2, sort="_score")
            results = query['hits']['hits']
            for bm25_movie in results:
                bm25_movie['_source']['query_score'] = bm25_movie['_score']

            return render_template('es_searchengine.html', form=form, response=results)

        # Clear the whole data in the new dataset
        elif form.clear_all.data:
            if es.indices.exists(index="bm25_dataset"):
                es.indices.delete("bm25_dataset")
            return render_template('es_searchengine.html', form=form)


    elif request.method == 'GET':
        return render_template('es_searchengine.html', form=form)

if __name__ == '__main__':
    app.run(debug=True)

BM25模型

该模型是Elasticsearch默认使用的，因此我们直接利用multi_match去查询query。

首先您需要了解BM25检索模型的打分方式，即对于所有documents的不同fields (即所有documents的title field、type field、director filed…)，分别进行打分得到一个分数，最后加在一起得到最后的query score (即BM25模型的最终分数)，下图可以方便您进行理解：

假如你只想制作一个简易的搜索引擎，那么你可以直接忽略BM25F模型相关的代码

BM25F模型

BM25F检索模型具有不同的打分方式，即我们将所有fields合成一个field，如下图的BM25F View所示即为我们需要打分的filed，因为每个电影的简介(movie_synopsis)字数不同，所以我们添加一定数量的None，以确保movie_synopsis中都含有100个字(超出100字的我们就不管了，因为不是很多)，这样做可以保证最后打分的公平性。

对于我们的SearchForm.py文件：

  • 为了在此搜索引擎中处理Web表单，我们使用Flask-WTF。
  • 对于需要其他HTML属性的字段，可以将其作为参数传递。
from flask_wtf import Form
from wtforms import SubmitField, StringField, SelectField
from wtforms.validators import DataRequired

class SearchTitleForms(Form):

    #对应页面的下拉栏
    select = SelectField(
        'select',
        [DataRequired()],
        choices=[
            ('None', 'None'),
            ('Title', 'title'),
            ('Type', 'type'),
            ('Score', 'score'),
            ('Director', 'director'),
            ('Actors', 'actors'),
            ('Synopsis', 'synopsis')
        ]
    )

    #以下分别对应各个按钮
    weight = StringField("Weight")

    dataset = SubmitField("Create Dataset")
    clear_all = SubmitField("Clear Dataset")

    q = StringField("Keyword")
    bm25_search = SubmitField("BM25 Search")
    bm25f_search = SubmitField("BM25F Search")

总结

本项目是在学习完Information Retrieval课程后，为了充分理解并使用学过的内容，因此用Python调用Elasticsearch的API，并穿插使用了一下Kibana，最后用Flask实现了一个简单的电影搜索引擎。

其中，BM25F模型，参考了论文：Simple BM25 Extension to Multiple Weighted Fields

  • 索引&检索模型：本设计直接使用了Elasticsearch中的内置的索引以及BM25检索方法，但是为了更好地理解，可以尝试不结合Elasticsearch，而直接用Python编写Index，各种检索model的方法，去实现搜索引擎。
  • Flask：我们的搜索引擎页面十分简洁，因为作者本人并没有学习过Flask相关用法，这些都是现学现卖而已，所以想美化页面的朋友们可以自行优化。另外，对于页面上手动Create/Clear Dataset，其实是可以去掉改成自动的，只需要在app.py文件中修改即可，只是为了更方便其他人对BM25F设计的逻辑进行理解。
